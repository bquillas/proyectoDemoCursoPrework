Clase 01

html - atributos - anidamiento

  1. html: El HTML (Hyper Text Maker Language), es un lenguaje de intepretación de texto.
  2. atributos: son las que van dentro en las etiquetas.
  3. Anidamiento: son la secuencia de las etiquetas.

Clase 02

HTML - Head - Body

  1. HTML: La etiqueta html es la más importante en una hoja de extension .html.
  2. Head: Esta etiqueta es la parte que no se muestra al renderizar, pero es una de las más importantes, ya que dentro de ella se encuentra las etiquetas (meta, title, link).
  3. Body: Esta etiqueta contiene todo lo que se va a mostrar al renderizar.
